# Subdactyl Dashboard

A production-ready web dashboard for Pterodactyl, allowing users to create free servers.

## Tech Stack
- **Frontend**: Vue 3, Vite, Vanilla CSS
- **Backend**: PHP (Laravel-style)
- **Database**: MariaDB / MySQL

## Setup
### Backend
1. Navigate to `backend/`
2. Run `composer install`
3. Copy `.env.example` to `.env`
4. Run `php artisan migrate`

### Frontend
1. Navigate to `frontend/`
2. Run `npm install`
3. Run `npm run dev`
